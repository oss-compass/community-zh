**01 领域画像新形象**

首先，来看看项目深度洞察页面的领域画像。

以往的文字描述可能有些枯燥，但现在，我们迎来了全新的展示方式！通过彩色条来展示不同领域的贡献，色条越长代表贡献越多。想了解详情？只需将鼠标悬停在上面，就能立即弹出浮窗显示贡献的具体信息。

深蓝色代表 Code 类贡献，浅蓝色是 Code 管理贡献，橙色是 Issue 类贡献，黄色是 Issue 管理类贡献。观察类贡献则用灰色条表示，只有选中右上角的访客贡献者才会显示这类贡献。

这种新的展示方式，让贡献的类型和数量一目了然，是不是很赞呢？快来体验一下吧！

**02 组织信息可修改**

除此之外，OSS Compass 还为大家带来了一个令人期待的功能更新：现在可以编辑每位贡献者的组织信息了！

只需点击贡献者详情表格中的组织信息修改图标，就可以在弹窗中进行修改。当然，这个权限只有开源项目管理者才拥有哦。

而对于个人想要修改自己的组织信息，也很简单，只需进入个人账号设置页面，在页面下方点击“新增组织信息”或者对已有组织信息进行修改即可。

目前组织信息默认显示英文，输入部分字母按提示选择就好了，非常方便快捷！

所以，不管是领域画像的更新还是组织信息的编辑，都是 OSS Compass 为了给大家提供更好的使用体验所作出的努力。
